Proiect C# (.NET 8) - Sistem de Management al Rezervărilor

Aceasta este o aplicație consolă C# dezvoltată pe platforma .NET 8, care simulează un sistem de bază pentru managementul rezervărilor la apartamente. Proiectul demonstrează principii fundamentale ale Programării Orientate pe Obiecte (OOP).

Modele Principale:

Apartament: Gestionează adresa, capacitatea, prețul și o listă a rezervărilor.

Rezervare: Conține detaliile clientului, perioada șederii și o listă de obiecte Persoana.

Persoana: O clasă simplă pentru a stoca numele și anul nașterii oaspeților.

Funcționalități: Aplicația rulează un meniu interactiv în consolă care permite utilizatorului:

Adăugarea unei noi rezervări la un apartament selectat.

Căutarea unei rezervări existente după numele clientului.

Listarea tuturor rezervărilor active, grupate pe apartament.
